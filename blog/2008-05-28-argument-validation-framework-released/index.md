## Examples:

**Validation pre conditions:**

```csharp
static string ValidationFunc(int a,string b,DateTime c)
{
  //pre conditions:
  a.Require("a")
   .IsGreaterThan(10);

  b.Require("b")
   .NotNull()
   .NotEmpty()
   .LongerThan(2)
   .StartsWith("Ro");

  c.Require("c")
   .IsInRange(new DateTime(2000, 01, 01),
   new DateTime(2010, 01, 01));
```

**Validation post conditions:**

```csharp
string res = "Foo";
//post condition:
return res.Require("res")
          .NotNull()
          .LongerThan(2)
          .ShorterThan(100);
```

**Async fork:**

```csharp
static int ForkIt()
{
   int a = 0;
   int b = 0;

   //begin async fork
   Fork.Begin()
       .Call(() => a = SomeSlowCall())
       .Call(() => b = SomeOtherSlowCall())
       .End();
   //async fork done
   return a + b;
}
```

**Flextensions:**

```csharp
//fluent casts, no need to wrap in ((int)var).

someVar.As<Boo>().Yoo();

//formatting and output;

"Hello {0} {1}".FormatWith(firstName,LastName).Output();

//string matching
```
